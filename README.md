# Karol Design Profissional - Sistema de Agendamento

Este é o sistema oficial de agendamento e portfólio para a profissional Karol Design, desenvolvido com foco em alta performance e estética moderna.

## 🚀 Funcionalidades
- **Agendamento Online:** Interface intuitiva para clientes.
- **Gestão de Serviços:** Seleção dinâmica (Cílios, Sobrancelhas, Limpeza de Pele).
- **Portfólio em Nuvem:** Galeria de fotos com compressão automática.
- **Efeito Neon RGB:** Design exclusivo com identidade visual marcante.

## 🛠️ Tecnologias
- Tailwind CSS (Design Responsivo)
- Lucide Icons (Interface Visual)
- JavaScript ES6+ (Lógica de Dados)
- Data SDK (Persistência em Nuvem)
